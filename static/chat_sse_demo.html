Como Funciona o SSE

O que é SSE?

Server-Sent Events (SSE) permite que o servidor envie dados em tempo real para o cliente. A resposta é enviada token por token, criando o efeito de "digitação" do ChatGPT.

Formato dos Eventos

data: {"type": "start", "session_id": "..."} data: {"type": "token", "content": "Olá"} data: {"type": "token", "content": ", "} data: {"type": "token", "content": "como"} data: {"type": "end", "processing_time_ms": 1234} data: [DONE]

Exemplo JavaScript

const response = await fetch('/chat/quick/simple-openai/stream', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ message: 'Olá!' }) }); const reader = response.body.getReader(); const decoder = new TextDecoder(); while (true) { const { done, value } = await reader.read(); if (done) break; const chunk = decoder.decode(value); // Processa os eventos SSE... }
